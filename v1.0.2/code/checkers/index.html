    """
    !!! summary "Summary"

        value (Any):

            `#!py True` if the value is of the specified type or one of the specified types; `#!py False` otherwise.

        ```{.py .python linenums="1" title="Prepare data"}

        >>> check_type = int

        >>> is_value_of_type(value, check_type)
